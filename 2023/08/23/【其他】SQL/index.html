基本SQL语句

  • DDL数据定义语言: Create, Alter, Drop, Rename, Truncate
  • DML数据操作语言: Insert, Delete, Update, Select
  • DCL数据控制语言: Commit, Rollback, Savepoint, Grant, Revoke

导入现有的数据库

  • source 文件路径
  • 图形化界面

基本Select语句

  • select * from xxx
  • select x as y from xxx ；y作为x别名
  • select x y from xxx ；没有逗号，y作为x别名，或者给y加双引号
  • select distinct x from xxx ；去掉重复行
  • Null参与运算结果都是Null，可以用 isnull(xx,0)来替换成0运算
  • 遇到和关键字重名的加着重号 ``
  • desc xxx ；查看表结构
  • select x from xxx where y = 1 ；过滤
  • select 100 + ‘1’ from xxx ;结果为101，字符串隐式转换为数值
  • 安全等于’<=>’可以对Null进行判断，Null <=> Null 返回1
  • select least(x,y) from xxx ；排序比如名字
  • select x from xxx where x like ‘%a%’ ；查询包括a字符的x
  • where x ory y ；异或，满足x不满足y，或满足y不满足x
  • select x from xxx order by y desc；降序排序 asc为升序排序
  • 列的别名只能在order by中使用，不能在where中使用
  • where要申明在from之后，order by之前
  • select x from xxx order by y desc, z asc ;先按y降序，再按z升序排列
  • select x from xxx limit 0,20 ;分页，首页显示前20条数据，20,20 显示20-40条数据
  • select x from xxx limit 2 offset 31 ；显示32和33的数据

多表查询

  • 笛卡尔积错误
  • select x, y, xxx.z_id from xxx, yyy where xxx.z_id = yyy.z_id ;加入连接条件，如果选择时要Z_id，必须指明哪个表
  • 等值连接 vs 非等值连接
    • 比如表A中Salary在表B中没有对应，但又low和high，则where A.salary betweeen B.low and B.high 就是非等值连接
  • 自连接 vs 非自连接
    • select t1.a_id, t1.x, t2.a_id, t2.x from xxx t1, xxx t2 where t1.b_id = t2.a_id ；xxx这张表自引用，a_id是unique id，b_id也是重复表示a_id
  • 内连接 vs 外连接
    • 内连接：select x, y from xxx inner join yyy on xxx.z_id = yyy.z_id
    • 外连接：left outer join 左外连接；right outer join右外连接； full outer join 满外连接 mysql不支持
    • 左中图：select x ,y from xxx left join yyy on xxx.z_id = yyy.z_id where yyy.z_id is null
    • 满外连接：
      • 左上图 union all 右中图
    • 右下图：
      • 左中图 union all 右上图

  • select x, y from xxx join yyy on xxx.z_id = yyy.z_id 可以用
  • select x, y from xxx natural join yyy 或
  • select x, y from xxx join yyy using (z_id) 来替代

单行函数

  • 数值函数：ABS(x)，SIGN(x)返回符号，LEAST(x,y,z…)最小值，GREATEST(X,y,z…)最大值，RAND(x)随机数，round(x,1)保留1位，trauncate(x,1)截取1位小数，conv(10,x,y)把x进制的10转换为y进制
  • 字符串函数： char_length(x), length(x) 返回字符串字节个数，concat(x,y) 连接，lpad(x,10,’*‘)在左边用*补足至10位，相当于右对齐效果, rpad同理左对齐，nullif(x,y)比较x,y，相等返回null，否则返回x
  • 日期时间类型函数：curdate(),curtime，当前日期或时间
    • select extract(second from now()) from dual ；从当前时间获得秒
    • select data_add(now(), interval 1 year) ;现在的时间加1年
  • 流程控制函数：
    • IF(x,y,z)如果x为true返回y，否则返回z，x为一个判断条件
    • case when x1 then y2 when x2 then y2 else y3 end z ;在列z记录不同情况
  • 加密解密函数：
    • select MD5(‘x’), SHA(‘x’) from dual

聚合函数

  • select avg(x) from xxx, sum() max() min()
  • group by 声明在where后面，order by前面
  • 如果过滤条件中使用了聚合函数，如max()等，必须用having而不能用where
  • select x max(y) from xxx group x having max(y)>1000
  • 聚合函数不能嵌套，单行函数可以嵌套

sql99语法

  • select … from … join … on … where… group by… having … order by … limit …
  • 执行顺序: from —- on —- left/right join —- where —- group by —- having —- select —- distinct —- order by —- limit

子查询

  • select x, y from xxx where y > (select y from…)
  • 多行子查询 in/any（存在就行）/all
  • 除了group by和limit之外，其他地方都可以声明子查询
  • where exist () ；搭配子查询，有记录就返回，停止本次子查询
