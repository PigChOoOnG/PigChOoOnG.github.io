有关中国经济的关键问题

有关中国经济的关键问题

经济波动问题：克强指数 vs Real GDP
经济结构问题：居民消费占比GDP偏低
经济政策问题：非金融部门（政府、居民及非金融企业）债务总额占GDP比重超250%

消费占比过低：S-I=NX，居民消费占比下降，储蓄占比上升，随之投资占比攀升，但大量投资压低回报率抑制投资意愿，导致2001年后净出口明显扩张。但遇到国际需求下降时，大量储蓄只能靠国内投资消化，但投资过剩严重，必须增加消费占比GDP比重，降低国内储蓄规模。

供给和需求的问题
价量反向变化，由需求主导；价量同向变化，由供给主导。观察中美GDP和通胀相关性变化，两国并不相同。

利用solow model分解中国GDP增速，从改革开放后，资本贡献持续上升，劳动力贡献持续下降，技术贡献在GFC之后下降明显。

Euler’s theorem: $Y = rK + wL$
Cobb-Douglas production funciton: $\frac{rK}{Y} = \frac{\frac{\partial Y}{\partial K}K}{Y} = α$, $\frac{wL}{Y} = \frac{\frac{\partial Y}{\partial L}L}{Y} =1-α$
Solow Model:
$K_{t+1} = sA_{t}F(K_{t}, L_{t})+(1-δ)K_{t}$
$k_{t+1} = \frac{sA_{t}f(k_{t})+(1-δ)k_{t}}{1+n}$
$k^ = (\frac{sA}{n+δ})^{\frac{α}{1-α}}$
$y^ = A(\frac{sA}{n+δ})^{\frac{α}{1-α}}$
$c^ = (1-s)y^ = A(1-s)(\frac{sA}{n+δ})^{\frac{α}{1-α}}$
$s^{golden} = α$

发展战略与经济绩效
在solow中，资本份额为α，同时由$y = Ak^{α}l^{1-α}$，其代表生产对资本的依赖程度，α越大，k越大，l越小。即α和资本与劳动力配比成正比，可以看做对资本密集度的刻画。

要研究的问题是，给定一个经济当前所拥有的资本和劳动力的资源禀赋结构，应该采用什么样资本密集度的生产技术。

在人均资本存量较低时，劳动密集型技术对经济更有利；而在人均资本存量很高时，资本密集型技术更有利。

新中国成立初期优先发展重工业，此时人均资本密集度还没到达资本密集型行业自发出现的阈值，重工业企业缺乏自生能力。政策压低利率和工资率，同时压低生活必需品价格。

均衡宏观经济学
总需求$Z = C+I+G+NX$, $C = c_{0}+c{1}Y$带入，市场出清时候$Z=Y$，可得$Y = \frac{1}{1-c_{1}}(c_{0}+I+G+NX)$。

之前$Y= AF(K,L)$是隐含生产不受需求的约束，即萨伊定律的意思，而这里隐含着存在需求不足的问题（凯恩斯的假设）。

现实中很难对$c_{1}$作出准确估计，根据Friedman持久收入假说，在一个简单两期模型中，消费者关注总效用。最大化条件得到$C_{1} = C_{2} = \frac{Y_{1}+Y_{2}}{2}$ 为未来平均收入预期，消费者消费只因持久收入而变化，而不对暂时收入变化作出反应。

消费与储蓄
中国居民消费偏低，储蓄偏高。

中国国内储蓄主要是由居民和非金融企业，其次是政府，以及金融企业。国际上通常企业储蓄和居民储蓄具有负相关关系，但中国同时高居民储蓄以及高企业储蓄。

基于模型的分析：
全社会总储蓄= $AF(k_0,L)-c_1$
居民部门储蓄= $r_1 k_{c0}+w_1 L+d_1-c_1=k_{c1}-k_{c0}$
企业部门储蓄=$AF(k_0,L)-w_1 L-r_1 k_{c0}-d_1=k_{f1}-k_{f0}$
总储蓄是收入-消费，净储蓄是总储蓄-投资。

为求得最优消费和投资比，我们假设消费者最大化效用$u(c_{1})+βu(c_{2})$，得到跨期边际效用比$\frac{u^{‘}(c_{1})}{βu^{‘}(c_{2})} = 1+r_{2}$，比较其和资本回报率就能评价消费投资比例是否合适，但现实中消费主观偏好无法测量，但模型得到$\frac{u^{‘}(c_{1})}{βu^{‘}(c_{2})} =\frac{d_{2}}{v_{1}}$，在居民拥有企业时，可以比较资本汇报率和股权收益率。

这个Ramsey模型隐含假设了Say’s law，“供给创造需求”是说不会出现生产过剩和就业不足。在高效的市场中，居民拥有企业所有权，经济中所有产出都会变成消费者的收入，并被花在消费和储蓄（投资）上，创造出和总产出相等的总需求。

假设效用函数CRRA：$U= \frac{c^{1-σ}_1}{1-σ}+β\frac{c^{1-σ}_2}{1-σ}$
生产函数Cobb-Douglas: $AF(k,l)= Ak^α l^{1-α}$

财政政策

凯恩斯的乘数效应：$Y = \frac{1}{1-c_1}(c_0-c_1 T+ I+G)$

李嘉图等价：在政府支出已经给定的前提下，税收还是发债并不影响消费者决策，也不影响宏观经济的总需求和总产出。

货币政策的分析

古典两分法：名义价格灵活变化，货币存量不影响实体经济变量（货币中性），货币增长率不影响实体经济变量（货币超中性）。

弗里德曼提出最优货币数量，认为名义利率衡量的货币的边际收益，而货币的边际成本可以视为0，则最优货币数量应让名义利率为0。

黏性价格下，$y-y_n = φ(π-π^e)$
奥肯法则：$y-y_n = -b(u-u_n)$
从而推出expectation-augmented phillips curve：$π-π^e = -φ(u-u_n)$

Time inconsistency：如果政府能够作出可信承诺，可以获得更高的社会福利。可信地承诺未来通胀为0，那么通货膨胀率就会比相机决策时低，而产出水平相同，提升全社会福利。

费雪方程式: $R = π + r = π + AF_1(k,l)$，即$R-π = AF_1(k,l)$，获取信贷资金的真实利率等于实物资本带来的回报率。我国之前R受管制，通胀上升时真实利率下降，实体经济信贷需求增加，刺激信贷增长导致经济更加过热。从而造成更大的经济波动。

流动性效应的模型显示，如果货币政策向实体经济传导途径流畅的话，会推高通胀，从而使名义利率上升；但如果传导途径不流畅，流入金融市场，推高股价，名义利率下降。2013年下开始社融长期同比少增，但银行间市场成交量大幅增加，体现金融体系资金泛滥。

伯南克对抗通缩非常规货币政策：

  • 在常规货币政策领域不断降低短期基准利率，直到利率达到零下限
  • 央行买入长期国债，压低长期利率（即QE）
  • 直接购买风险资产，如MBS等，压缩风险溢价
  • 大规模减税、扩张财政，大量发行货币，货币财政化来弥补

开放经济中的货币

汇率平价关系：$\frac{1+R_d}{1+π_d} = \frac{1+R_f}{1+π_f}$
取对数近似后，得到无抛补利率平价: $\frac{δE_d}{E_d} = R_d - R_f = π_d - π_f$
利率平价比一价定律更普遍，就算一价定律不成立，无抛补利率平价也会因为国内外资本的自由流动而成立。

在灵活价格下，货币政策传导完全顺畅时，本国基础货币总量突然跳高，价格水平也跳高。通胀预期保持0不变，名义利率由费雪方程式决定也不变，本币汇率则一次性跳高并稳定。

但实际中，价格水平是逐渐走高，通胀预期也会先上升再下降回到0，名义利率由费雪方程式和流动性效应共同决定，开始流动性效应导致名义利率短期内下降，随着货币政策传导推进，通胀走低，名义利率慢慢上升到扩张前的水平。

蒙代尔不可能三角即，本国货币政策独立性（自由设定R_d）、固定汇率和资本自由流动三个目标最多同时满足两个。但在经常账户盈余国家，面对不断的外汇流入，可以通过外汇冲销来同时达到三个目标的，但面临资本流出时，不可能三角是硬约束。

思想理论发展

  • classical：1776亚当斯密发表‘国富论’开始，代表人物有David Ricardo, Thomas Malthus, John Stuart Mill，相信商品价格由成本决定
  • neoclassical：18世纪下半叶-20世纪初，代表人物有Cournot, Jevons, Walras, Marshall，认识到商品价格由供需共同决定

这两阶段统称为古典理论，相信市场的高效，Say’s law认为供给创造需求，

  • neoclassical synthesis: Keynes认为经济完全有可能陷入有效需求不足、产能过剩的局面，需要政府管理需求。

Hayek则认为政府不可能得到足够多的准确信息来对资源作出有效配置。此外，政府在经济中的作为越多，越容易导致公权力对个人自由的侵犯，最终导致集权主义。

萨缪尔森提出neoclassical synthesis，市场有高效运转的时候，也有失灵的时候。核心分析框架是IS-LM-AS-AD模型，和求解最优化问题的微观经济学很不一样。

  • new classical: Lucas批判，对菲利普斯曲线的消失反映了当人的预期发生变化，宏观经济变量之间的关系会不稳固，因此必须从微观主体的理性出发，求解最优化问题来推导宏观的稳固关系式。

Ramsey模型中重现了经济均衡与鲁滨逊经济的等价、萨伊定律、古典两分法等。

Prescott和Kydland发展了RBC模型，在Ramsey模型中加入随机变化的技术冲击，得到结论经济周期是市场对外生技术冲击的最优反应，而不是市场失灵。在经济衰退时因为技术进步率低，真实工资率低，工人主动减少工作，失业是自然的。但这很难解释大萧条时那么高的失业率是工人自己主动选择。

  • new keynsian：假设种种名义刚性阻碍价格灵活调整，使得市场面对外部冲击时，短时间无法作出最优反应。同样短期也无法对政府政策变化做出完全反应，因而无法完全对冲政策影响。所以政策短期内对经济有影响，可以来弥补市场对外部冲击的不完全反应。

  • new neoclassical synthesis：结合RBC和new keynsian，现代主流宏观经济模型的框架。
