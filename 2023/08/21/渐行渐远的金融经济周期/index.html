渐行渐远的金融&经济周期（彭文生）

发表于 2023-08-21 分类于 Macro

…

关于classical, keynes/ newclassical, new keynes的总结：
古典经济学包括一般均衡论和货币数量论两个部分。一般均衡论解释商品的相对价格和由此引导的资源配置，货币数量论解释商品的货币价格和一般价格水平。

  • 一般均衡论从微观沿生到宏观可以用Say’s law概括：供给创造需求，经济不会出现生产过剩的情况，资源被充分利用，失业是自愿的，是在work与leisure之间的选择，外在的冲击只会导致短暂的波动，市场最终能回到充分就业。
  • 货币数量论 MV=PQ，强调货币的交易价值，

凯恩斯主义核心是总供给理论：总需求决定总供给，大萧条是有效需求不足，投资者信心下降，股市下跌，融资条件趋紧，企业减少投资，工人失业，消费降低，导致另一部分生产的工人失业。

  • 凯恩斯的货币理论强调货币储值功能，人们有流动性偏好来自未来的不确定性。流动性偏好的不稳定性，导致利率发生超预期变化影响总需求，货币对实体经济的影响起码短期不是中性。为应对有效需求不足，政府应提高支出来弥补私人部门支出不足；为应对高利率，应当直接购买长债，因短债利率接近0的情况下类似货币，购买短债整体流动性没有增加。
  • 总需求变化先影响就业，充分就业后才影响物价。发展出Phillips Curve
  • 1960s后期，弗里德曼提出自然失业率，对应潜在增长率，即和货币、通胀这些名义变量无关；但同时也认同经济增长于潜在增长的差距影响通胀。

新古典主义依然强调市场配置资源的有效性和政府干预的有害性，Lucas提出理性预期假设，好比解释不同时间点所有商品供给和需求的均衡，发展出RBC Model，试图说明经济周期波动的总需求管理政策在短期内也不能改善民众福利，否定政府逆周期操作。

新凯恩斯主义用古典经济学的微观基础，加入理性预期，但强调市场中有摩擦，导致暂时的不平衡，假设价格不能及时变动。货币对增长和失业有短期的影响，但长期是中性的。

分析框架：

  • 长期总供给曲线：潜在GDP和其驱动因素（劳动力、资本存量、全要素生产率）的关系
  • 短期总供给曲线：描述CPI通胀率和总需求压力
  • 货币政策
  • 财政政策
  • 国际收支

操作标的上，1970-1980s主要央行以货币总量为中间目标，假设基础货币和广义货币，以及广义货币和名义GDP/通胀之间的稳定关系。1990s央行转以短期利率为操作标的，依赖Taylor rule(US)，上述框架的效果是1987-2007的The Great Moderation.
金融危机后的反思，主流分析框架有三个局限

  • 对资产价格的变动及其影响把握不够
  • 对货币信用周期的理解
  • 对制度变化的关注

人口变化：
过去20年的人口结构变化，一是人口红利，总人口中劳动年龄人口占比增加，抚养比下降，人口红利明显；二是城乡转移。其他国家都是先完成城镇化，再出现人口红利，而在中国两者重叠。未来人口红利逐渐消失以及农村富余劳动力大幅减少，缩小未富先老（人口老龄化向发达国家趋同的速度 - 人均收入趋同的速度）的缺口更紧迫。
对发展中国家而言，城镇化中农村劳动力向城镇转移，提高了平均的劳动生产率或全要素生产率，Lewis描述工业部门相对传统农业部门扩张的两个阶段，第一阶段劳动力无限供给，劳动力过剩，工资取决于维持生活所需的生活资料的价值；第二阶段劳动力短缺，农业部门剩余劳动力全部被工业部门吸收，工资取决于劳动的边际生产力。其中的转折点被称作‘刘易斯转折点’。一般来讲，人口红利阶段，消费需求相对较弱，投资需求相对较强，国内需求相对较弱，出口相对较强。另一方面，劳动人口占总人口比例上升，储蓄率上升，转化为投资，增加未来资本存量从而促进经济的潜在增长。
在人口红利的阶段，生产能力超过本身消费的环境中，价格上升对供给的拉动作用效率较高，总体物价上升压力较小。在人口红利见顶回落的调整阶段，劳动力供给趋紧，工资增长速度相对其他价格更快，支持消费但减少了企业利润从而抑制投资，消费品价格相对投资品价格上升较快。
传统的现收现付的退休机制，人口红利更早的发达国家进行了一系列制度改革，包括延长退休年龄、增加劳动参与率（主要是提高劳动年龄的妇女的劳动参与率）、鼓励储蓄，来减轻年轻人的纳税负担。另一种养老安排是储蓄积累制，减少当期消费来购买资产，通过资产积累提供养老保障。在人口红利阶段，储蓄者多于负储蓄者，竞争有限的投资机会，推高资产价格，人口红利消退的阶段，下一代用来购买老一代资产的储蓄减少。

增长：长期看供给，短期看需求
已进入长周期放缓的阶段，加入WTO的制度红利已经释放，人口结构朝着不利于经济增长的方向发展，房地产泡沫挤压实体经济。上世纪80年代开始的几次快速增长，主要是全要素增长率的增长。

  • WTO的制度红利已经释放，对外开放度（外贸总额/GDP）近年的回落。对出口的依赖下降并不能直接说经济有效地再平衡了，而要同时深化内部市场改革和增强国内市场竞争度，才能减少对中长期供给的影响。
  • 城乡劳动力的转移空间减小，一个线索是低端劳动力工资的上升速度，反映部分劳动力的供应。人口红利（生产者/消费者)见顶，将引导储蓄率下降，投资率下降，资本存量增速也将下降。
  • 房地产价格快速上升对其他实体经济的挤压。房地产投资快速扩张，短期拉动需求，但造成土地价格上涨，土地的有效供给持续不足会影响供给能力的增长。

短周期波动：从三驾马车分析

  • 政策逆周期和政治周期：政府换届和五年规划对基建投资实际增长率的影响。
  • 财政主导的政策，短期对需求刺激较高，但负面影响一是财政赤字扩大增加货币需求，利率上升影响私人部门融资条件；二是往往带来对银行信贷的直接占用，增加民营和中小企业融资的难度和成本。

结构失衡与再平衡：
产出（供给）在劳动力和资本之间分配，政府政策在其中起到重要的再分配作用。企业部门（资本）的收益转化为投资，居民内部收入（劳动力）差距扩大压低了平均消费倾向，提高了储蓄率。
依照此逻辑，收入分配失衡是症结，导致总体消费率下降，储蓄率上升，带来高投资和出口，进而产业结构偏工业、制造业，与消费相关的服务业发展落后。但比较国外收入差距扩大但消费仍强劲，这只能是原因之一。

  • 居民部门：可支配收入包括居民获得劳动报酬、经营性收入、财产性收入以及政府的经常性转移收入，可支配收入在居民消费与储蓄之间分配，储蓄转化为居民投资
  • 企业部门：增加值和财产收入扣除劳动者报酬、生产税和转移支付后，就是企业的可支配收入，全部转化为企业储蓄和投资
  • 政府部门：生产税、所得税、社保缴款等扣除社保支出、生产性补贴、国债利息等，转化为政府消费和政府储蓄，储蓄进一步转化为投资性拨款与政府自身固定资产投资

居民消费率 = $\frac{居民消费}{GDP} = \frac{居民消费}{居民可支配收入} \frac{居民可支配收入}{GDP} = (1-储蓄率)收入占比$
居民消费率的下降的解释：

  • 收入占比下降的重要原因为工资增速低于劳动生产率。人口红利有利于劳动生产率的提高，同时劳动供给的竞争抑制工资增长，叠加城乡转移，劳动力从第一产业转移到第二、三产业，整体生产率提高，但农村大量隐形失业人口导致工资增长缓慢。
  • 储蓄率上升原因是人口红利时期青壮年人数较多，平均储蓄率上升。
  • 企业对居民部门的挤压：金融市场发展落后时期，金融企业与能获得信贷的企业部门可以使用居民部门的便宜资金，导致居民储蓄存款利率低于资金的影子价格，压低居民可支配收入，抑制消费。
  • 政府对居民部门的挤压：政府重投资、轻转移支付与公共服务，不能有效发挥调节收入分配的功能。
  • 居民部门内部收入差距扩大：我国税收以流转税（增值税、营业税、消费税、关税）占大头，税基是交易额，本质是消费税，低收入者因消费占收入比重大而税负偏重。

货币信用周期（这部分没很清楚）：
首先描述流动性偏好：$ M_2 = b* W $ 给定b流动性偏好一定的条件下，央行增加流动性，利率下降，私人部门资产配置提高风险资产比重，推动后者价格上升，改善私人部门融资条件，同时通过财富效应刺激消费。金融危机后，美国私人部门b下降，如果流动性总供给不增加，风险资产价格将下跌。联储购买长期国债和ABS投放货币，以及卖短买长扭曲操作，在短期利率接近0的情况下，短债也相当于流动性。
流动性偏好理论描述的是货币的储值功能，货币数量论描述的是货币的交易功能。弗里德曼的货币数量论其实是两者的结合，认为货币需求取决于

  • 人们对总体财富（永久收入）
  • 决定资产配置的债券利率和预期的股市回报率（持有货币的机会成本）
  • 预期的商品和服务价格通胀率

第三点是和流动性偏好理论的差别，加入了预期，则在通胀预期上升时，除了购买风险资产还会购买耐用消费品，增加总需求进而影响通胀。即货币供应多了一个影响总需求，进而影响通胀的渠道。
货币信用周期最核心的两个指标是银行信用和房地产价格，前者代表融资条件，后者代表投资者对风险的认知程度。发生的频率比一般商业周期小，持续的时间更长。货币心信用周期的顶点往往和金融危机联系在一起。

房地产泡沫：
第一个讨论了高房价收入比的问题。如果要可持续的话，一是未来居民收入增长要显著快于房价增长，或者继续放大居民部门购房的金融杠杆。但细分来看年青一代的杠杆高于平均水平的，加上老龄化发展，老一代对年轻一代的支持也会下降。
房价租金比则部分反映了投资房地产的回报率，当然还要考虑实际的住房持有成本。
高空置率是另一个关注点。
过去十几年我国人口红利与劳动力城乡转移相叠加，储蓄率上升很快。资本账户管制对外投资受限，支持了房价的快速上涨和股市的扩容。

汇率问题：
我国私人部门对外负债，政府部门对外资产。2007年之前强制结售汇，之后虽然放开，但由于人民币升值预期，私人部门不愿持有外汇，央行大量购汇，人民币投放增加，带动银行信贷。低风险的货币资产快速增长促使私人部门重新配置资产，增加对风险资产的需求，推动风险资产价格上涨，强化了外资流入和私人部门持有人民币的动机。

渐行渐远的金融周期

货币不是中性的：现代版的货币中性论，新凯恩斯假设价格不能及时调整，货币变动在短期对总需求产生影响。但主张央行只把货币政策作为逆周期调节的工具。货币的量是内生的，央行货币政策通过控制短期利率，影响中长期利率和资产价格，将总需求控制在经济的潜在产出水平附近。
用流动性偏好理论（$M_2= b * W$）解释金融危机后美国$M_2/GDP$大幅上升，但CPI维持低位的现象：私人部门流动性偏好（货币需求）上升，风险偏好降低，如果流动性供给不相应增加，则风险资产价格将大跌，联储通过QE和扭曲操作增加流动性。
解释我国$M_2/GDP$的上升并未带来恶性通胀，流动性增加人们风险偏好上升，配置更多风险资产尤其房产，房价上升。假设房价下跌，风险偏好下降，对流动性需求上升，利率上升，增长下行，商品价格有通缩压力。
顺周期的机制：经济景气，人们预期收入增加，贷款把未来收入转化为现在的购买力，支持当前消费。企业利润率增加，给定资金成本利率，通过贷款投资扩大生产规模增加利润。
明斯基理论，从资产负债表出发，将经济分为家庭、非金融农业、金融机构和政府四个部门。债务人分为三种类型：

  • hedge：资产产生的现金流足以覆盖负债带来的现在和未来的利息和本金偿付。
  • speculative：现金流足以支付致债务的利息，但不足以偿还本金，需要依靠债务展期来维持，银行是此类。
  • ponzi：现金流不能覆盖本金和利息，如房地产开发金融，现金流在完成销售后才产生，虽然可以通过发新债来维持利息偿付，净资产会随之下降，即对债权人的保障下降，其通过融资将利息资本化的空间越来越小。

繁荣初期，以hedge为主，随着获利增加，人们变得乐观，更愿意通过负债获得资产，信用和房地产价格相互促进，speculative和ponzi的比例提高，整个体系的脆弱性增加，一旦融资成本上升（可能是政策紧缩或者外生冲击），speculative变成ponzi，原本ponzi的净资产蒸发，债务人被迫减少开支，抛售资产还债，资产泡沫破裂（明斯基时刻），信用紧缩，经济步入衰退期。
奥地利学派强调的是结构的视角，假设从均衡出发，银行信用扩张，一部分获得信用的企业购买设备和中间品增加投资，用于生产消费品的资源部分被引导到生产投资品，引起投资品价格相对消费品价格上升。一段时间内投资拉动GPD增长，呈现经济繁荣景象。这个过程中消费者时间偏好和储蓄率都没有变化，银行信贷给企业家带来储蓄增加的假象从而增加投资。时间推移，消费品供应不足，消费品价格开始上升，消费者时间偏好增加，更注重当前消费，导致储蓄率下降，均衡利率上升，引导资源回到消费品行业，投资行业呈现产能过剩。
中国信贷周期的形成和强化，透过银行体系的资产负债表，$M_2$对应的资产可以归为三大类：对家庭和企业部门的债券（信贷）、对政府债券、持有的境外资产（外汇占款）。进一步区分央行和商业银行，从21世纪初到2016年，央行总资产对GDP比例从40%上升到2009年高点超过60%，然后下跌。而商业银行资产对GDP比例从200%一路上升到290%。美国则相反，金融危机前商行对GDP比例持续上升，危机后几年显著下降，央行对GDP比例危机危机前平稳，危机后大幅上升。总结为金融危机后美国紧信用、松货币，中国宽信用、紧货币。

地根和银根：
土地的金融属性：估值无客观基础、对利率敏感、交易依赖外部融资。
我国过去十几年房价快速上升的宏观背景是：1995-2000年生产者数量超过消费者，劳动力从农村向城市转移。

  • 对住房的消费需求增加
  • 高储蓄率与资本账户管制相结合，对国内资产的需求增加
  • 过剩型经济使得CPI通胀率和波动性下降

信用扩张起到了推动作用：土地和房产为抵押品的贷款，包括部分地方政府融资平台的负债、企业以制造业或者其他用途申请的贷款但实际投入房地产行业、影子银行包括理财产品筹集的资金通过银行表外渠道进入房地产行业。

从2011年开始，民间投资增速下滑成为趋势性现象，有人认为还是各种显性隐形的限制导致，玻璃门、弹簧门、旋转门三重门。更关键的是房地产价格持续上涨挤压了制造业等实体经济的发展空间，而制造业中民间投资占比高。
